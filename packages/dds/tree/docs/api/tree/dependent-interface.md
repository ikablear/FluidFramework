### markInvalid {#markinvalid-methodsignature}

Invalidates the `Dependent`<!-- -->.

This is typically called by a [Dependee](/docs/apis/tree/dependee-interface) this `Dependent` registered its dependance on (via [Dependee.registerDependent()](/docs/apis/tree/dependee-interface#registerdependent-methodsignature)<!-- -->) when the dependee is invalidated. This is how invalidation propagates to the dependance graph along Dependee-to-Dependent edges.
