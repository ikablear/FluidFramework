## Method Details

### compose {#compose-methodsignature}

Compose a collection of changesets into a single one. Every child included in the composed change must be the result of a call to `composeChild`<!-- -->, and should be tagged with the revision of its parent change. Children which were the result of an earlier call to `composeChild` should be tagged with undefined revision if later passed as an argument to `composeChild`<!-- -->. See [ChangeRebaser](/docs/apis/tree/changerebaser-interface) for more details.

#### Signature {#compose-signature}

```typescript
compose(changes: TaggedChange<TChangeset>[], composeChild: NodeChangeComposer, genId: IdAllocator): TChangeset;
```
