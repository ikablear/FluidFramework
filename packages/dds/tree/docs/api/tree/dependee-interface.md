Interface for object which can change and invalidate [Dependent](/docs/apis/tree/dependent-interface)<!-- -->s when changing. Tracks a set of `Dependent`<!-- -->s on which [Dependent.markInvalid()](/docs/apis/tree/dependent-interface#markinvalid-methodsignature) will be called when the output of this changes.

Dependencies are only used for invalidation, so there is no need to make a `Dependee` when the source data is immutable.

#### Returns {#registerdependent-returns}

true iff dependent was added (false if it was already tracked or otherwise not needed to be added). If true is returned, a removeDependent can be performed to remove dependent.

<b>Return type:</b> boolean

### removeDependent {#removedependent-methodsignature}

Removes a dependent which was previously added by a call to registerDependent which returned true. This causes dependent's markInvalid to no longer get called when this dependee is invalidated.
