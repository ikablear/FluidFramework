## Property Details

### anchors {#anchors-propertysignature}

Set of anchors this forest is tracking.

To keep these anchors usable, this AnchorSet must be updated / rebased for any changes made to the forest. It is the responsibility of the called of the forest editing methods to do this, not the forest itself. The caller performs these updates because it has more semantic knowledge about the edits, which can be needed to update the anchors in a semantically optimal way.

#### Signature {#anchors-signature}

```typescript
readonly anchors: AnchorSet;
```
