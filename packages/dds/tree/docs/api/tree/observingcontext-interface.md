## Property Details

### dependent {#dependent-propertysignature}

The computation which is running in this context. When this computation observes a [Dependee](/docs/apis/tree/dependee-interface)<!-- -->, it should be recorded here via [recordDependency()](/docs/apis/tree#recorddependency-function)<!-- -->.

This dependency tracking allows these dependees to invalidate this dependent in the future via [Dependent.markInvalid()](/docs/apis/tree/dependent-interface#markinvalid-methodsignature)<!-- -->.

It not provided, this observer does not need dependency tracking.
