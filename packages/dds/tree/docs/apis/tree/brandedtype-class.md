Helper for [Brand](/docs/apis/tree#brand-typealias)<!-- -->. This is split out into its own as thats the only way to: - have doc comments for the field. - make the field protected (so you don't accidentally try and read it). - get nominal typing (so types produced without using this class can never be assignable to it). - allow use as [Opaque](/docs/apis/tree#opaque-typealias) branded type (not assignable to `ValueType`<!-- -->, but captures `ValueType`<!-- -->).

See [MakeNominal](/docs/apis/tree/makenominal-interface) for some more details.

Do not use this class with `instanceof`<!-- -->: this will always be false at runtime, but the compiler may think its true in some cases.
